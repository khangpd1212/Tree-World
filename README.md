## cách lấy code

git clone (tên repository)

## checkout nhánh

git checkout (nhánh)

## xem coi có chuyển nhánh chưa

git branch

## add file

git add .

## commit

git commit -m "ghi cai gi do"

## pull từ main về nhánh

git pull origin main

## push lên nè

git push origin (nhánh)
